이벤트 설명

《산타》 자유부대의 두번째 이벤트

부대원들과 더 친밀한 관계가 되고 싶다면
이벤트에 참여해보시는건 어떨까요?

다양한 이벤트에 참여해서 부대원들과 즐거운 시간도 보내고
총 40,000,000 길 상당의 상품도 얻어보자!

이벤트 일정

  • 이벤트 날짜 : 2020년 11월 7일 (토) 오후 9시 (약 1시간 반 진행 예정)
  • 이벤트 장소 : 산타 자유부대 하우징 앞마당

참가상

푹신푹신 방석 (25만길 상당)

OX 퀴즈

이벤트 설명

  1. 운영진이 내는 문제를 듣고 O / X 중 선택
  2. 최후 5인에게 길 지급 (남은 인원 중 한 명이라도 경쟁 원할 시, 다음 문제 진행)
  3. 전원 탈락 시, 전체 인원 처음부터 재진행

우승 상품

최후 5인 이하 인당 [전체 상금 6,000,000 길 / 남은 인원 수]
예시 전체 상금 6,000,000 길 / 최후 인원 4인 = 인당 1,500,000 길

패션쇼

이벤트 설명

  1. 디스코드 [패션쇼 참가 신청] 채널에 양식에 맞춰서 신청
  2. 자신의 차례에 무대에 올라가서 패션쇼, 컨셉 소개 및 어필
  3. 전체 인원 참가 완료 후, 부대 디스코드에서 투표
  4. 심사는 《산타》, 《무전》은 0~2점, 부대원 투표 인당 1점씩 계산, 자기 자신 투표 가능
  5. 득표 순위 1~3등 보상 지급

우승 상품 (순위 높은 순 선택)

라자한 모직 가을 셔츠 (250만길 상당)
퀘인트렐 러플 드레스 + 비옷 + 선원의 셔츠 (200만길 상당)
순백색, 칠흑색 염료 중 택 2 (100만길 상당)

사진 콘테스트

이벤트 설명

  1. 4인 1팀으로 팀 구성
  2. 제한 시간 내 파판 모든 맵(던전 포함)에서 자유롭게 사진 촬영
  3. 가장 잘 나온 사진 1장을 부대 디스코드 [사진 콘테스트] 채널에 제출 (팀당 1명만)
  4. 심사는 《산타》, 《무전》은 0~2점, 부대원 투표 인당 1점씩 계산, 자기 팀 투표 가능
  5. 득표 순위 1, 2등 팀 전원 보상 지급

우승 상품 (사다리타기, 순위 높은 팀부터 주사위 높은 팀원부터 선택)

라자한 모직 가을 셔츠 (250만길 상당)
부대원의 휘장복 (100만길 상당)
우피티 뿔피리 (100만길 상당)
꼬마친구 세트 (100만길 상당)
백색증 검은양 뿔피리 (100만길 상당)
다양한 옷 세트 (220만길 상당)
반역자 외투 (125만길 상당)
비옷 (70만길 상당)
초과 팀원
500,000 길

숨바꼭질

이벤트 설명

  1. 에오르제아 전역에 숨어있는 《산타》, 《무전》, 《띵글이》 찾기
  2. 자신과 운영진이 함께 나오게 스크린샷 찍기 (닉네임 보이게)
  3. 부대 디스코드에 제출 (선착순 2명)

우승 상품

[1등] 30일 이용권 (선물)
[2등] 환상약
