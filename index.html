이벤트 설명

《산타》 자유부대의 첫 이벤트

부대원들과 더 친밀한 관계가 되고 싶다면
이벤트에 참여해보시는건 어떨까요?

다양한 이벤트에 참여해서 부대원들과 즐거운 시간도 보내고
총 3,000만길 상당의 상품도 얻어보자!

이벤트 일정

  • 이벤트 날짜 : 2020년 5월 24일 (일) 오후 9시 (약 1시간 반 진행 예정)
  • 이벤트 장소 : 산타 자유부대 하우징 앞마당

점프킹 : 너에게 닿기를

이벤트 설명

  1. 이벤트 당일 운영진이 지정한 탑의 꼭대기에 도달하라!
  2. 꼭대기의 《산타》에게 거래 신청 (선착순 3명)

우승 상품

1등 3,000,000 길
2등 1,500,000 길
3등 주사위 값 x 1,500 길 (최소 0 길 ~ 최대 ‭1,498,500‬ 길)

얼음상 대회 : 진 시바와 함께

이벤트 설명

  1. 진 시바 토벌전에서 페이즈 전환(얼어붙는 순간) 때 파티 전체가 보이는 스크린샷 찍기
  2. 최소 6인 이상 파티만 참여 가능
  3. 부대 디스코드에 제출하고 투표
  4. 심사는 《산타》, 《무전》, 《사장》은 0~3점, 부대원 투표 1점씩 계산, 자기 팀 투표 가능
  5. 우승 팀 전원 보상 지급 (단, 주사위 높은 순 선택. 상품 소진 시 길 지급)

우승 상품 (주사위 높은 순 선택)

우피티 뿔피리 (400만길 상당)
장인의 룩템 세트 2개 [현장 작업복 or 민소매셔츠 + 작업바지] (200만길 상당)
꼬마 백기사 (100만길 상당)
헤어 카탈로그: 전설 (100만길 상당)
연기 교본: 기대기 (100만길 상당)
상품 소진 시 (100만길)

숨바꼭질 : 운영진을 찾아라

이벤트 설명

  1. 에오르제아 전역(신생 맵만 해당)에 숨어있는 《산타》, 《무전》, 《사장》을 찾기
  2. 자신과 운영진이 함께 나오게 스크린샷 찍기 (닉네임 보이게)
  3. 부대 디스코드에 제출 (선착순 1명)

우승 상품 (1등만 지급)

30일 이용권 (선물)
